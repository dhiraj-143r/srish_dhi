Dashboard Agents

Watcher Agent

Email Parser

Parses incoming emails from AgentMail webhooks. Fetches full message content, thread context, and attachment metadata.

AgentMail SDKParsing

Triage Agent

LLM Classifier

Classifies urgency, category, and action using GPT-4o with structured JSON output and chain-of-thought reasoning.

GPT-4oJSON ModeCoT

Research Agent

Web Scraper

Researches sender background by scraping company websites and email URLs using Firecrawl. Generates LLM summary.

FirecrawlGPT-4o

Vision Agent

Image Classifier

Analyzes email attachments using Roboflow computer vision. Classifies images as invoices, receipts, documents, etc.

RoboflowCLIP

Drafter Agent

Reply Generator

Generates contextual, tone-appropriate email replies using triage analysis, research findings, and attachment context.

GPT-4oContext Fusion

WEBHOOK CONFIGURATION

Register a webhook URL to receive AgentMail events

1

Start ngrok tunnel

ngrok http 8000
2

Copy the HTTPS URL

Use the Forwarding URL that looks like https://xxxx.ngrok-free.app

3

Register the webhook

curl -X POST http://localhost:8000/api/webhook/register \
  -H "Content-Type: application/json" \
  -d '{"url": "https://YOUR-NGROK-URL/webhook/email"}'
4

Send a test email
